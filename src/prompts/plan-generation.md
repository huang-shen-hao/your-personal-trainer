# 训练计划生成 Prompt

## 任务说明

根据用户档案（身体状况、训练经验、目标、器械条件等）生成个性化训练计划。

## 输入信息

从用户上下文中提取以下信息：

### 基础信息
- 年龄、性别、身高、体重
- BMI 和体脂率（如有）
- 训练经验水平（无/初级/中级/高级）

### 训练目标
- 主要目标（增肌/减脂/力量提升/体能提升/综合/康复）
- 次要目标（如有）

### 训练条件
- 器械条件（无器械/家用器械/完整健身房）
- 训练场所（家/健身房/户外）
- 每周可训练天数
- 单次训练时长偏好

### 健康状况
- 伤病史（需要避免的动作或部位）
- 体态问题（如有）
- 其他健康限制

## 生成规则

### 1. 经验水平适配

#### 无经验/初级（< 6 个月）
- **分化方式**：全身训练（Full Body）
- **频率**：2-3 次/周
- **动作选择**：基础复合动作为主，少量孤立动作
- **组数×次数**：2-3 组 × 10-12 次
- **强度**：RPE 6-7（中等）
- **学习重点**：动作模式、技术标准、身心连接

**典型动作库**：
- 深蹲（杠铃/哑铃/高脚杯）
- 硬拉（罗马尼亚硬拉/哑铃硬拉）
- 卧推（杠铃/哑铃/俯卧撑）
- 肩推（哑铃/杠铃）
- 划船（杠铃/哑铃/坐姿划船）
- 引体向上/高位下拉
- 平板支撑、卷腹

#### 中级（6-18 个月）
- **分化方式**：上下肢分化（Upper/Lower）或推拉腿（Push/Pull/Legs）
- **频率**：3-5 次/周
- **动作选择**：复合动作 + 孤立动作，开始引入变式
- **组数×次数**：3-4 组 × 8-12 次
- **强度**：RPE 7-8
- **学习重点**：渐进超负荷、周期化、技术精进

**分化示例（上下肢）**：
- **上肢日**：卧推、肩推、划船、下拉、二头弯举、三头下压
- **下肢日**：深蹲、罗马尼亚硬拉、腿举、腿弯举、提踵、核心

#### 高级（18 个月以上）
- **分化方式**：推拉腿（PPL）或五分化（Bro Split）
- **频率**：4-6 次/周
- **动作选择**：复合动作 + 多样化孤立动作，高级技巧
- **组数×次数**：4-5 组 × 根据周期变化
- **强度**：RPE 8-9，周期化波动
- **学习重点**：高级技巧（drop set, rest-pause, 超级组）、薄弱环节针对、周期化

### 2. 目标适配

#### 增肌（Hypertrophy）
- **容量**：每肌群每周 10-20 组
- **强度**：中等重量（70-85% 1RM）
- **次数**：8-12 次/组
- **休息**：60-90 秒
- **节奏**：2-0-1-0（2秒离心，无停顿，1秒向心，无停顿）
- **技巧**：机械张力 + 代谢压力，泵感训练

#### 减脂（Fat Loss）
- **容量**：保持肌肉量的最小有效容量（每肌群 6-12 组/周）
- **强度**：中等重量（70-80% 1RM）
- **次数**：10-15 次/组
- **休息**：45-60 秒（或循环训练）
- **附加**：HIIT 或 LISS 有氧（每周 2-4 次）
- **重点**：保持力量，避免肌肉流失

#### 力量提升（Strength）
- **容量**：每肌群每周 6-12 组
- **强度**：大重量（80-90% 1RM）
- **次数**：3-6 次/组
- **休息**：3-5 分钟（充分恢复）
- **动作**：三大项为主（深蹲、硬拉、卧推）+ 辅助动作
- **技巧**：周期化（波动训练量和强度）

#### 体能提升（Conditioning）
- **容量**：适中
- **强度**：中低重量，高次数或时间（15-20 次或 30-60 秒）
- **休息**：短休息（30-45 秒）或循环训练
- **动作**：功能性复合动作，全身参与
- **附加**：有氧训练为主（跑步、游泳、HIIT）

### 3. 器械适配

#### 无器械（徒手）
- **动作替代**：
  - 深蹲 → 深蹲、箭步蹲、单腿深蹲
  - 卧推 → 俯卧撑、窄距俯卧撑、下斜俯卧撑
  - 硬拉 → 早安式、单腿硬拉（平衡）
  - 划船 → 反向划船（需横杆）、超人式
  - 肩推 → 倒立撑（进阶）、Pike 俯卧撑
- **进阶方式**：增加次数、减慢节奏、单侧训练、增加杠杆难度

#### 家用器械（哑铃/弹力带）
- **动作替代**：
  - 杠铃深蹲 → 哑铃深蹲、高脚杯深蹲
  - 杠铃卧推 → 哑铃卧推、地板卧推
  - 杠铃硬拉 → 哑铃硬拉、罗马尼亚硬拉
  - 引体向上 → 弹力带辅助引体向上、哑铃划船
  - 腿举 → 哑铃深蹲、保加利亚分腿蹲
- **优势**：更大活动范围、单侧训练、灵活性高

#### 完整健身房
- **动作选择**：全部动作可用
- **器械组合**：自由重量（主菜）+ 固定器械（辅助）
- **优化训练**：利用不同器械的特点（如龙门架的恒定张力）

### 4. 安全性与个性化

#### 伤病适配
- **膝盖问题**：
  - 避免：深度深蹲、跳跃、冲击性动作
  - 替代：箱式深蹲（控制深度）、腿举（脚位高）、游泳
  - 强化：股四头肌 VMO、臀中肌

- **腰背问题**：
  - 避免：传统硬拉、过顶深蹲、脊柱过度负荷
  - 替代：罗马尼亚硬拉（轻重量）、高脚杯深蹲、核心稳定训练
  - 强化：核心肌群、臀大肌

- **肩部问题**：
  - 避免：颈后推举、过度内旋动作、大重量卧推
  - 替代：中立握、哑铃动作（活动度更自由）、肩袖强化
  - 强化：肩袖肌群、肩胛骨稳定肌

#### 体态适配
- **圆肩/驼背**：
  - 增加：背部训练容量（划船、面拉、YTW）
  - 减少：胸部训练容量（或暂停）
  - 拉伸：胸部、前三角肌

- **骨盆前倾**：
  - 增加：核心训练（死虫、鸟狗、平板）、臀大肌训练
  - 拉伸：髂腰肌、股四头肌、竖脊肌

### 5. 周期化设计

#### 线性周期化（Linear Periodization）
适合初学者

```
第 1-4 周：适应期（轻重量，高次数，学习动作）
第 5-8 周：肥大期（中等重量，8-12 次）
第 9-12 周：力量期（大重量，4-6 次）
第 13 周：Deload 减量周
```

#### 波动周期化（Undulating Periodization）
适合中高级

```
周一：重量日（3-5 次，RPE 8-9）
周三：容量日（8-12 次，RPE 7-8）
周五：轻量日（12-15 次，RPE 6-7）
```

#### Deload Week（减量周）
- **频率**：每 3-6 周安排 1 次
- **目的**：恢复疲劳、预防过度训练
- **方式**：
  - 容量减少 50%（减少组数）
  - 或强度减少 20-30%（降低重量）
  - 或两者结合

### 恢复建议

**睡眠**：每晚 7-9 小时，尤其是训练日

**主动恢复**：
- 低强度有氧：散步、游泳（20-30 分钟）
- 拉伸或瑜伽（每周 1-2 次）

**恢复工具**：
- 泡沫轴：训练后放松肌筋膜
- 按摩球：针对深层肌肉
- 冰敷/热敷：急性疼痛用冰，慢性紧张用热

**Deload Week**：
- 第 [X] 周为减量周
- 训练量减少 50% 或强度降低 20-30%
- 目的：恢复疲劳，预防过度训练

### 进阶与调整

#### 渐进超负荷策略

**第 1-2 周**：适应动作，完善技术
**第 3-4 周**：每周增加 2.5-5% 重量
**第 5-6 周**：继续增加重量或增加次数
**第 7-8 周**：引入高级技巧（drop set, rest-pause）
**第 9 周**：Deload（减量周）
**第 10-12 周**：重新评估，调整计划

#### 何时调整计划

**增加重量/次数**：
- 如果连续 2 周能超额完成目标次数（如目标 3×10，完成 3×12）
- 如果 RPE 持续低于目标（如目标 RPE 8，实际 RPE 6）

**减少容量/强度**：
- 如果出现持续疲劳、睡眠质量下降
- 如果 RPE 持续过高（目标 RPE 8，实际 RPE 9-10）
- 如果力量下降或停滞超过 2 周

**更换动作**：
- 如果某个动作持续导致不适或疼痛
- 如果进步停滞，尝试动作变式
- 每 8-12 周可以更换部分辅助动作以保持新鲜感

### 训练记录与监控

**每次训练记录**：
- 动作、组数、次数、重量
- RPE（主观用力感，1-10）
- 训练时长
- 整体感受

**每周评估**：
- 训练完成度（是否按计划完成）
- 平均 RPE
- 体重变化
- 睡眠质量
- 整体状态

**月度评估**：
- 力量进步（PR 记录）
- 体重/体脂变化
- 体围变化（如有测量）
- 照片对比
- 计划调整需求

### 注意事项

**重要提醒**：
- ⚠️ 如有任何疼痛或不适，立即停止训练并咨询医生
- ⚠️ 初学者应在专业教练指导下学习动作技术
- ⚠️ 训练强度应循序渐进，避免过度训练
- ⚠️ 营养和恢复与训练同等重要

**个性化调整**：
- 本计划基于您的当前状况设计，但需要根据实际反馈调整
- 如果感到过度疲劳，增加休息日或降低训练量
- 如果进步缓慢，检查营养、睡眠和训练强度

## 生成注意事项

1. **因人而异**：严格基于用户的实际情况（经验、目标、条件、伤病）
2. **可操作性**：动作说明要清晰，用户能独立完成
3. **安全第一**：充分考虑安全性，给出警告和替代方案
4. **渐进原则**：不要一开始就过度训练
5. **完整性**：包含热身、主训练、放松，以及营养和恢复建议
6. **灵活性**：提供调整和进阶的指导
7. **鼓励性**：语气积极，增强用户信心
